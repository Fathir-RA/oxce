# Projek Pemrograman Berbasis Objek

## OC-XE

**OC-XE** singkatan dari **Ocean Expedition** yang merupakan aplikasi pemesenan tiket menyelam yang memudahkan dalam memesan tiket, melihat berbagai lokasi dan informasi.
